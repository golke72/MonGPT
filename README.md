# 🤖 MonGPT — Telegram бот с 37 функциями

**MonGPT** — это мощный Telegram бот с искусственным интеллектом, который умеет всё: от простого общения до генерации картинок, поиска музыки и сканирования QR-кодов.

## 🚀 **37 ВОЗМОЖНОСТЕЙ:**

### 🧠 **ОСНОВНЫЕ (1-10)**
1. 🤖 Умный чат с ИИ (молодежный сленг, мат)
2. 🧠 Память диалога (помнит 15 сообщений)
3. 🎤 Голосовые сообщения → текст
4. 🎥 Кружки (видеосообщения) → текст
5. 🎬 Видеофайлы → аудио → текст
6. 🎵 Аудио и музыка → распознавание речи
7. 🌐 Поиск в интернете (погода, курсы, новости)
8. 🔗 Открывает и пересказывает ссылки
9. 📌 Закреп сообщений в группах
10. 💬 Ответы с цитированием (reply)

### 🎨 **КРЕАТИВ (11-20)**
11. 🎨 Генерация картинок по тексту
12. 🎥 Конвертация видео в кружки
13. 🔲 Сканирование QR-кодов из фото
14. 🔳 Генерация QR-кодов
15. 🎭 Создание стикеров из фото
16. 📈 Графики курса валют
17. 📊 Графики криптовалют
18. 🔐 Генератор надежных паролей
19. 🖼️ Улучшение качества фото
20. 🎨 Стилизация фотографий

### 👑 **ТВОЁ ЛИЧНОЕ (21-30)**
21. 💰 Твой баланс 666.666.666 токенов
22. 📊 Статистика использования
23. 🔄 Команда /add_tokens (только для тебя)
24. 📱 Нижняя клавиатура с кнопками
25. 🎯 Инлайн-кнопки под ответами
26. 🧹 Команда /clear для сброса истории
27. 🗣️ Голосовой ответ (текст → аудио)
28. ⏳ Анимация загрузки (⏳)
29. ⚡ Кэширование частых запросов
30. 🚀 Быстрые ответы (0.3-0.8 сек)

### 🎵 **МУЗЫКА (31-37)**
31. 🔍 Поиск музыки по названию
32. 📎 Скачивание MP3 с YouTube
33. 🖼️ Добавление обложек в теги
34. 🎛️ Выбор качества (128/192/320 kbps)
35. 🧹 Автоматическая очистка файлов
36. 📝 Добавление метаданных (ID3)
37. 🎵 Формирование плейлистов
